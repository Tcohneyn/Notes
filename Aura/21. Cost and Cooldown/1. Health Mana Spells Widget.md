这份视频教程主要讲解了如何为 RPG 游戏设计并搭建一个更复杂的 **HUD（抬头显示）** 框架。作者通过大量使用 **Box（框组件）** 来进行精确的 UI 布局，为后续添加经验条（XP）、主动技能和被动技能图标做准备。

以下是全部内容的标题大纲式详细总结：

### 一、 新阶段目标：RPG 核心机制与 HUD 升级 [00:00 - 01:00]

- **课程进度说明**：地牢场景初具规模，现在进入下一个重要阶段：增加经验值（XP）、新法术和升级系统等 RPG 元素。
- **HUD 现状**：目前的两个球体（血量和法力）太单调，需要整合更多信息。

### 二、 HUD 设计蓝图：布局与功能规划 [01:00 - 02:16]

- **视觉方案**：
  - 左右两侧各有一个大球体（血量、法力）。
  - 中间区域包含：装备的主动技能图标（显示对应按键）、被动技能图标。
  - 底部或中央区域将包含 XP 经验条。
- **技术实现思路**：通过嵌套大量的水平框（Horizontal Box）和垂直框（Vertical Box）来精确控制尺寸。

### 三、 创建主 UI 组件及基础架构 [02:16 - 04:15]

- **创建资源**：在 `UI/Overlay/SubWidgets` 文件夹下创建 `WBP_HealthManaSpells`。
- **设置尺寸**：将填充屏幕（Fill Screen）改为 **Custom（自定义）**，尺寸设为 **1141 x 216**。
- **基础布局（Base Box）**：
  - 添加一个 **Overlay** 作为根节点（Overlay Route）。
  - 添加一个 **Horizontal Box（水平框）** 并命名为 **Base Box**。
  - **主三分区**：在 Base Box 内添加三个水平填充框：
    1. **Health Box**（填充比例 0.2）：放置血球。
    2. **Central Box**（填充比例 0.6）：放置技能和经验信息。
    3. **Mana Box**（填充比例 0.2）：放置法力球。

### 四、 中央区域（Central Box）的细分布局 [04:15 - 07:33]

- **左右切分**：Central Box 内放入两个垂直框：
  - **Offensive Box**（填充比例 0.9）：用于主动进攻技能。
  - **Passive Box**（填充比例 0.1）：用于被动技能。
- **主动技能区（Offensive Box）纵向划分**：
  1. **Above Icons Box**（0.3）：放置文本标签和按键提示。
  2. **Spell Icons/Globes Box**（0.45）：放置技能图标。
  3. **Spacer Box**（0.3）：底部占位空间。
- **标签区细化**：在 Above Icons Box 内添加两个框：
  - **Offensive Text**：显示“Offensive”文字。
  - **Input Text**：显示按键提示文字（LMB, RMB, 1-4）。

### 五、 集成现有的血量与法力球体 [07:33 - 09:00]

- **球体组件**：
  - 将 `WBP_HealthGlobe` 拖入 Health Box，设置为 **Fill（填充）**。
  - 将法力球拖入 Mana Box，设置为 **Fill**。
- **视觉优化**：
  - 修改血球属性：将进度条（Progress Bar）和重影条（Ghost Bar）的默认百分比设为 **100%**。
  - 确保血球显示为红色，法力球显示为蓝色。

### 六、 文本标签与按键指示器的细节配置 [09:00 - 13:40]

- **进攻/被动标签**：
  - 字体设为 **Amaranth Regular**，大小为 **14**，添加 1 像素描边。
  - 进攻文本颜色设为暗红色，居中对齐。
- **按键提示（Input Text）布局**：
  - 在 Input Text 框内添加 **6 个水平子框**（分别对应左键 LMB、右键 RMB、数字键 1、2、3、4）。
  - 所有的文本设为白色，并重命名（如 `Text_LMB`, `Text_1` 等），确保它们在图标上方均匀分布。

### 七、 技能占位框与被动技能区搭建 [13:40 - 15:55]

- **主动技能槽位**：
  - 在 **Spell Globes Box** 内添加 **6 个垂直框**，分别命名为 `SpellBox_LMB` 到 `SpellBox_4`。
  - 全部设为 **Fill** 模式，等待后续填充具体的技能图标组件。
- **被动技能槽位**：
  - 在 Passive Box 内添加三个垂直框：`PassiveBox_1`、`PassiveBox_2` 和一个 **Spacer（占位框）**。
  - 设置填充比例使图标看起来接近正方形。

### 八、 总结与后续预告 [15:55 - 结束]

- **当前进度**：完成了一个极其详细的 HUD 盒子架构图，涵盖了 6 个主动技能位和被动技能位。
- **下一步**：由于目前的过程比较繁杂（Tedious），下一节将创建一个专门用于显示技能图标的小型球体组件（Globe Widget），并将其填入这些预留的框中。
