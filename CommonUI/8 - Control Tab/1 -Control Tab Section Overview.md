## 🎮 键位重映射功能开发大纲

### 1. ⚙️ C++ 后端设置与数据查询

- **输入上下文注册（Input Context Registration）：**
  - 注册用于输入键的 **Input Mapping Context**。
- **键位查询与过滤：**
  - 从 C++ 查询所有 **Mappable Keys**（可映射键）。
  - 研究用于过滤键类型（如键盘键和手柄键）的方法。
- **数据对象创建：**
  - 创建 **Key Remap Data Object**，并研究如何通过代码构建它。

### 2. 🎨 UI 与虚拟绑定（UI & Virtual Bindings）

- **创建新的 Entry Widget：**
  - 为键位重映射功能创建一个新的列表项组件（Entry Widget）。
- **处理虚拟绑定（Virtual Bindings）：**
  - 处理所需的虚函数绑定，以确保 UI 能够正确响应输入和数据显示。

### 3. ⌨️ 键位重映射核心逻辑

- **监听按键输入：**
  - 研究如何使用 **Input Preprocessor** 从代码中监听和捕获用户按下的输入键。
- **键类型过滤与通知：**
  - 一旦成功过滤出所需的键类型（例如，只接受键盘键或手柄键），将通知 **Key Remap Data Object**。
  - 由数据对象处理 **实际的键位重映射逻辑**。

### 4. ✅ 辅助功能与错误处理

- **输入显示：**
  - 使用 **Bell Icons**（图标）来显示当前映射的键盘键和手柄键。
- **映射重置：**
  - 实现键位映射的 **重置逻辑**（将键位恢复到默认设置）。
- **警告提示：**
  - 实现警告屏幕的显示逻辑，例如当用户试图将键位重置为一个已是默认值的键时，显示警告。
