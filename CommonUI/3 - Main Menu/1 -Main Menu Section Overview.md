## 🚀 新章节概述：主菜单 UI 与游戏手柄支持

本章节将围绕主菜单 UI 的创建和增强展开，重点是实现完整的游戏手柄交互支持和模块化的确认屏幕。

### I. 核心 UI 组件开发

- **菜单按钮基类**：创建所有菜单按钮的 Native C++ 基类。
- **主菜单 Widget**：创建主菜单 Widget。
- **输入监听**：实现在“按下任意键”界面中监听输入按键事件。
- **样式资产**：创建按钮和文本的不同样式资产。

### II. 按钮交互与信息展示

- **描述文本广播**：在按钮悬停时，广播该按钮的描述文本。
- **绑定动作按钮**：添加绑定动作按钮，显示手柄上的对应按键。
- **键位映射**：学习如何通过数据表中的 Common Input Data 设置按键映射和图标映射。

### III. 高级 UI 功能与支持

- **游戏手柄支持**：确保 UI 对游戏手柄的完整支持。
- **确认屏幕**：
  - 创建模块化的 **确认屏幕** (`Confirmation Screen`)。
  - 创建新的 **异步动作** (`Async Action`) 节点，用于在蓝图中推送确认屏幕。

### IV. 内容屏幕集成

- **故事屏幕**：在点击“故事”按钮后，添加新的故事屏幕。
